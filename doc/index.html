asdf.us/ascii documentation

These are some handy documents which address some of the more obscure features of the asdf.us color code tool: * tips.txt - Tips on using the keyboard * irssi.txt - Instructions on using IRSSI to make color codes. * nopaint.txt - A guide to "No Paint" Documents on Shaders * shadetut.txt - A brief tutorial on ASCII shaders. * shaders/brush.txt - Shaders designed to work on the brush * shaders/canvas.txt - Shaders designed to work on the canvas * shaders/util.txt - Miscellaneous utilities / snippets For more information on IRC, Color Codes, and much more, visit the documentation sitemap, part of the Jollo IRC Network. asdf.us/ascii
